# SocialMedia - Real-Time Social Media Web Application

## 📌 Project Overview

**SocialConnect** is a feature-rich social media web application built with **ASP.NET Core** and **SQL Server**. It allows users to connect with friends, send/receive friend requests, chat in real-time, and create group conversations. Real-time functionalities are powered by **SignalR**, ensuring instant message delivery and live updates.

---

## 🚀 Features

### ✅ User Authentication & Profile Management
- Register/Login using ASP.NET Core Identity
- Update profile information (Name, Profile Picture, Bio)

### ✅ Friend System
- Send friend requests to other users
- Accept or decline friend requests
- View pending requests
- See list of confirmed friends

### ✅ Real-Time Messaging (SignalR)
- One-to-one chat between confirmed friends
- Real-time message delivery with typing indicators
- Message read receipts (optional)

### ✅ Group Chats
- Create groups with confirmed friends
- Add/remove members from groups
- Real-time group messaging

### ✅ Notifications
- Real-time notifications for new friend requests
- In-app notifications for new messages
